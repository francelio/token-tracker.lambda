-----------------------------
Detailed Description 
-----------------------------
Token let's you keep track of your game's players, their score, who's winning, and more. With Token you can add players to your game by assigning them a color token. Just have your players pick a color, add them to your game, and start playing. Keeping score is easy as asking Token to add or subtract points for a color token. You can also ask Token to tell you whose winning, who has the lowest score, and what's the average score among all tokens. Asking Token to reset everyone's score is al it takes to start a new game from scratch. Here's a quick look at the features you'll get with Token:

- Lots of token colors to choose
- Tracking for each token's score
- Add and remove points for each token
- High score tracking
- Lowest score announcement
- Get the average score for the current game
- Add and remove points from all tokens at once
- Announce all tokens and their current score
- Lifetime data storage for subscribers!

Track your points using any of these colors for your player's token:
white, yellow, blue, red, green, black, brown, azure, ivory, teal, silver, purple, gray, orange, maroon, charcoal, aquamarine, coral, fuchsia, wheat, lime, crimson, khaki, magenta, olden, plum, olive, and cyan

Here's a breakdown of Token's features and some sample commands you can use with this skill:

Add a Token
Alexa, tell Token to add the color blue.

Delete a Token
Alexa, tell Token to remove the color blue.

Give a Token points
Alexa, tell Token to give three points to blue.

Give a Token a single point
Alexa, tell Token to give one to blue.

Subtract points from a Token
Alexa, tell Token to take two points away from blue.

Subtract a single point from a Token
Alexa, tell Token to take one from blue.

Get a Token's points
Alexa, tell Token to get the score for blue.

Give points to all Tokens
Alexa, tell Token to give everyone three points.

Subtract points from all Tokens
Alexa, tell Token to subtract three points from everyone.

Reset the points for all Tokens
Alexa, tell Token to reset all of the tokens.

Delete all Tokens
Alexa, tell Token to delete all of the tokens.

Get the highest score
Alexa, ask Token who has the highest score?

Get the lowest score
Alexa, ask Token who has the lowest score?

Get the average score
Alexa, ask Token what is the average score?

Announce all Tokens' points
Alexa, tell Token to read all the scores.

List all Tokens
Alexa, ask Token who is playing?

Announce the number of Tokens
Alexa, ask Token how many tokens there are.
